StockSharp

Бесплатные программы для
торговли и торговых роботов

StockSharp (кратко S#) - бесплатные программы для торговли на
всех площадках мира (российские, американские, азиатские;
акции, фьючерсы, опционы, биткоины, форекс и т.д.). Вы
сможете торговать в ручном режиме или с помощью торговых
роботов (обычные или HFT).

S#.DATA

Программа для автоматического скачивания исторических и реал-тайм маркет-данных:

Поддерживает массу источников: (РТС, ММВБ, Финам, Yahoo Finance, Quik, Transaq, SmartCOM, AlfaDirect, Plaza 2 (CGate), FIX/FAST, Micex Bridge, LMAX, Rithmic, Fusion, Interactive Brokers, OpenECry, Sterling, IQFeed, E*Trade, BTCE, BitStamp и многое другое). Высокая степенью сжатия (2 байта на сделку, 7 байтов на стакан). Любые типа данных (свечи, тики, стаканы, ордер-лог, опционы, новости и другое).

Программный доступ через API.

Экспорт в csv, excel, xml или базу данных..

Импорт данных из csv.

Задачи по расписанию.

S#.api

Библиотека для профессиональной разработки торговых роботов на языке C#. Для тех, кто программирует в Visual Studio, и является профессиональным программистом в области алготрейдинга.

Все наши продукты:

S#.Designer

универсальная программа для создания торговых роботов и стратегий.

S#.data

программа для скачивания маркет-данных.

S#.api

библиотека для написания торговых роботов на языке C# в Visual Studio.

ЛЧИ Viewer

графики конкурса ЛЧИ со сделками участников.

S#.shell

готовая графическая оболочка с исходными кодами.

S#.matlab

интеграция MatLab с торговыми системами. Торговля из скриптов MatLab.

S#.terminal

торговый терминал.
